# Atividade-final-modulo-II
Atividade final do segundo modulo, utilizando recursos bootstrap e flexbox


## Critérios para o desenvolvimento da aplicação

1 - Toda lógica da aplicação deve ser
desenvolvida em JavaScript

2 - Deve ser utilizado animations em, ao
menos, uma das páginas da aplicação

3 - A aplicação deve ser responsiva

4 - Deve ser utilizado os recursos do
Bootstrap no desenvolvimento da
aplicação
